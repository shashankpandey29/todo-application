# Full Stack Todo List Application

A modern and responsive **Full Stack Todo List Application** built using **Spring Boot**, **Thymeleaf**, and **Bootstrap 5**. This project demonstrates CRUD operations, MVC architecture, server-side rendering, and clean UI design for task management.

This project showcases:

- Full-stack web development with Spring Boot
- MVC Architecture
- CRUD Operations

##  Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/todo-list-application.git
cd todo-list-application
```

### 2️⃣ Configure Database

Update your `application.properties` file:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/todo_db
spring.datasource.username=root
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update
```

### 3️⃣ Build Project

```bash
mvn clean install
```

### 4️⃣ Run Application

```bash
mvn spring-boot:run
```

### 5️⃣ Open Browser

```text
http://localhost:8080
```
